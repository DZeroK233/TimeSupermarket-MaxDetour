# 时光超市最优布局求解器

基于 **贪心阻断 + 模拟退火** 的超市货架布局优化工具，用于最大化顾客路径上的货架接触面数（销量）。

## 问题抽象

- 网格地图，固定入口和出口。
- 可在空地上放置货架（1×1、1×2、2×2），货架必须**面向路径**才能售货。
- 顾客沿**最短路径**从入口走到出口。
- 目标：通过摆放货架和障碍物，使**路径总接触面数**（即销量）最大。

## 求解方法

### 1. 贪心初始化（Phase 1）
- 从空地图开始，反复计算当前最短路径。
- 选择路径上**能最大程度延长路径**的格子，将其设为障碍物。
- 重复直到无法继续提升路径长度，得到一组初始障碍物布局。

### 2. 模拟退火优化（Phase 2）
- 在初始解基础上，随机增加/删除/替换障碍物。
- 每次变动后重新计算最短路径，并调用**2D 装箱评估**：
  - 在路径外的空地上，贪心摆放允许尺寸的货架。
  - 每个货架根据相邻路径格子数，自动选择最佳宝石（1面/2面/4面/8面）。
  - 目标函数 = **总销量 × 1000 + 路径长度**（优先优化销量）。
- 以一定概率接受较差解，逐步降温，在时限内搜索更优布局。

### 3. 评估函数
- 对给定路径和障碍物，扫描所有可放置货架的位置。
- 按“单位面积销量”排序，贪心放置不重叠的货架。
- 记录每个路径格子被售货的次数（用于可视化）。

## 运行环境

- Python 3.6+
- 依赖：tkinter（标准库）

## 使用方法

1. 运行打开 GUI 窗口。
2. 左侧面板：
   - 选择工具（入口/出口/高台/橡皮擦），在网格上绘制基础地图。
   - 勾选允许的货架尺寸（1×1、1×2、2×2）。
   - 勾选“使用多面宝石”（8面/4面/2面宝石）。
   - 设置推演时间（秒）。
3. 点击 **“开始求最优解”**，等待计算完成。
4. 结果将自动绘制在网格上（金色路径、蓝色货架、棕色障碍物），右侧文本框显示详细统计。

## 注意事项

- 地图行列数建议 5~50。
- 必须至少有一个入口和一个出口。
- 求解过程中可点击“强制结束并输出最优”提前终止。
